# Mega menu — estructura propuesta para Shopify

Estructura que se importará a Matrixify (archivo `Embler-Mega-Menu.xlsx`). Valida antes de subir.

**Reglas:**
- Marca (nivel 1): SIN link (placeholder `#`).
- Categoria (nivel 2): link a la collection `brand-group`.
- Subcategoria (nivel 3): link a la collection `brand-group-subgroup`, máximo 3 por categoria (orden alfabetico).

**Marcas borradas del menu actual:** Sprinter, Citroën, Peugeot, Renault (no tienen collections).

**Total items en el menu:** 299 (13 marcas + 91 categorias + 195 subcategorias)

**Marcas agregadas:** Bentley, Fiat, Jaguar, Land Rover, Seat, Volvo.
